# ECommerce-App
ECommerce application with web api in c# and dot net core 5.0 

Using:

1) Dot net Core
2) Code First Migration
3) SQL Server Database
4) Angular
5) Bootstrap
6) Auth Guard
7) JWT Token
8) Angular Reactive Forms
9) Forms Validation
10) Interceptor
11) Redis Basket for adding item in Carts
